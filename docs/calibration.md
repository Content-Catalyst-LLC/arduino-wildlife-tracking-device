# Calibration Guide

## Why Calibration Matters

Wildlife telemetry performance depends not only on code, but on field conditions and hardware behavior.

Important variables include:

- GPS signal quality
- antenna orientation
- enclosure material
- wake interval
- battery voltage stability
- SD card write reliability

## Practical Calibration Notes

1. Test GPS acquisition outdoors
2. Measure average time to first valid fix
3. Compare logged coordinates against known locations
4. Verify that repeated logging events create valid records
5. Tune wake intervals based on desired balance between spatial resolution and battery life

## Notes

This prototype is best understood as a low-power telemetry reference design rather than a production wildlife tracking collar.
